Db Calls

With the $[(ConfigLinkCall)DbQueryCall /] one run queries on the serverside adding the result to EO.

In the following examples the $[(ConfigLinkCall)h2:mem:basic:AnObject configType="DbSqlConfig" /] is used.

It's just a query for the $[(ConfigLinkCall)AnObject /] table which will be initialized by $[(ConfigLinkCall)h2:mem:basic:Create configType="DbSqlConfig" /] when the spring boot container is started by $[(GithubLinkCall)spring||Start.java configType="NONE"/].

This is an in memory example which will be flushed whenever the server restarts. There are two other configurations involved:

  • The Database Configuration $[(ConfigLinkCall)h2:mem:basic configType="HostConfig" /]
  • The Sql init sequence $[(ConfigLinkCall)h2:mem:basic:Create configType="DbSqlConfig" /]
