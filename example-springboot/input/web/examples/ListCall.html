List Calls

With the $[(ConfigLinkCall)CsvSimpleReadCall /] one can read simple character separated files. It just split for a certain delimiter.

In the following examples the file $[(ConfigLinkCall)AnObject.csv configType="FileConfig" /] is used as data source.

The result in a table
