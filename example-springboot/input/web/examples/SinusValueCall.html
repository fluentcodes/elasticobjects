Compute Sinus

With the ==>{GithubLinkCall->ModelConfig, SinusValueCall}. a sinus can be computed. The example read the value from ==>{ConfigLinkCall->FieldConfig, sourcePath}. source and set the ==>{ConfigLinkCall->FieldConfig, targetPath}. target value with the sinus value.

You can change the value of source or the path itself to see it really works. "_serializationType" with value "STANDARD" will return a non typed JSON.
