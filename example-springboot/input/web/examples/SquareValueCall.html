SquareValueCall

#{GithubLinkCall->ModelConfig, SquareValueCall}. compute cos, sin, tan and toDegree Math functions depending on #{ConfigLinkCall->FieldConfig, function}. value. Default is "toDegree".

In the example toDegree will calculated from #{ConfigLinkCall->FieldConfig, sourcePath}. "squareDegree". The result will set in the #{ConfigLinkCall->FieldConfig, targetPath}. "value".
