Accessing An Excel File

The following example sets the content of \ ==>{GithubLinkCall->NONE, SPI|data.lists|AnObject.xlsx}. to ==>{ConfigLinkCall->FieldConfig, targetPath}. values via ==>{GithubLinkCall->ModelConfig, XlsxReadCall}..

The configKey refers to a configuration in ==>{GithubLinkCall->NONE, SPR||FileConfig.json#L2}. You're allowed to read the sheet, because your role "guest" is within the read permissions.

If you click, you will see the content is set to "values" as a map using the first row as keys.

If you remove "_serializationType" you can see the hidden fields used by EO, for example the duration of this call.
