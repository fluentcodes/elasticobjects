Is this Website Static?

The url of this page is /faq/ApplicationElasticObjectsOrg.html. This looks static. But it's dynamic with a spring boot mvc web end point defined in $[(GithubLinkCall)SP|web|WebEoGet.java configType="NONE"/].

By the RequestMapping "/examples/{selectedItem:.+}" the createFaqPage method is triggered which calls the createPage method. Here on the server side a new EO object is created and set some parameters like selectedItem or roles.

Core parameter when execute EO is the call $[(GithubLinkCall)TemplateResourceCall /] with the template $[(GithubLinkCall)SPI|web|ContentPage.html configType="NONE"/] for rendering.

The same EO object can be send by the generic interface $[(GithubLinkCall)SP|web|WebEo.java configType="NONE"/] by a post method. At the end of each page the code and the used elements/configuration for this are under "This Page Elements".

This site is an application example for an application documentation where practicability of EO will be checked.
