What is a Typed Path?

In the setter one can embed a type directive. This creates a typed object, when a configuration in $[(GithubLinkCall)EOR||ModelConfig.json configType="NONE"/] is defined.

Here a $[(ConfigLinkCall)AnObject /] is created in "a/b" and within this the field value $[(ConfigLinkCall)myString configType="FieldConfig"/] is set.
