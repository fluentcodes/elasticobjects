What is a configured call?

When it comes to calls using resources on the server, the information needed are also stored in configuration to allow access via a simple name.

There are two types of configurations:

  • FileConfig.json for file system
    access
  • DbSqlConfig.json for database access

A configured call has a configuration key within, where the configuration is connected. The configuration implements some methods to access a file, a list or execute some sql.

These configuration extends ConfigResources class, which contains some role access definitions.
