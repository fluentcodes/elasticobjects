An Arc Call

It's a bit boring defining a new call wrapper foreach Math arc method like in #{ConfigLinkCall->ModelConfig, CosinusValueCall}.

A more generic approach is implemented in #{GithubLinkCall->ModelConfig, SquareValueCall}. using the field #{ConfigLinkCall->FieldConfig, function}. to choose the Math method.

Beside to add an #{ConfigLinkCall->ModelConfig, SquareValueCall}. entry to #{GithubLinkCall->NONE, SPR||ModelConfig.json}. one has also add the field #{ConfigLinkCall->FieldConfig, function}. to #{GithubLinkCall->NONE, SPR||FieldConfig.json}.

Possible function values
  • sin
  • cos
  • tan
  • toDegree (default)
