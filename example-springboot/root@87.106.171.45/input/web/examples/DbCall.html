Db Calls

With the DbQueryCall one run queries on the serverside adding the result to EO.

In the following examples the h2:mem:basic:BasicTest is used.

It's just a query for the BasicTest table which will be initialized by h2:mem:basic:Create when the spring boot container starts.

This is an in memory example which will be flushed whenever the server restarts. There are two other configurations involved:

  • Database Configuration
  • Host Configuration
